## 專案狀態

目前僅保留核心專案說明，避免將非專案內容（例如編輯器快捷鍵或介面提示文字）誤放入 README。

## Sasha Deployment Protocol 規格檔

目前倉庫提供兩種可機器讀取格式，內容等價：

- `Sasha.Deployment.Protocol.vOmega.yaml`
- `Sasha.Deployment.Protocol.vOmega.json`

用途：
- 保留 `Sasha.Deployment.Protocol.vΩ` 的完整語義結構（含 Unicode key 與多行區塊）。
- 方便不同工具鏈以 YAML 或 JSON 載入。
